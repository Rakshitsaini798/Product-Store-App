# Product-Store-App
1. Project Description

The Product Store App is a web application that allows users to browse and explore products using real-time data from a public API. The application provides an interactive interface where users can search, filter, and sort products efficiently.

This project demonstrates the use of JavaScript for API integration, array higher-order functions, and dynamic UI rendering.

2. Purpose
Practice working with APIs using fetch
Implement search, filtering, and sorting using JavaScript
Build a responsive and user-friendly interface
3. API Used
Fake Store API

Base URL:
https://fakestoreapi.com

Important Endpoints
Get all products
https://fakestoreapi.com/products
Get single product
https://fakestoreapi.com/products/1
Get all categories
https://fakestoreapi.com/products/categories
Get products by category
https://fakestoreapi.com/products/category/electronics
Sort products (API level)
https://fakestoreapi.com/products?sort=desc
4. Planned Features
Core Features
Search Products
Users can search products by name using keywords
Filter by Category
Users can filter products based on categories
Sort Products
Users can sort products by price (low to high / high to low)
Product Display
Products will be displayed in a card format with image, title, and price
Optional Features
Favorites (using localStorage)
Dark Mode toggle
